Note: features that rely on Shift and Ctrl will not work properly on internal pages such as Speed Dial or Settings. Additionally, switching to one of these tabs while holding Shift or Ctrl, then releasing the key will confuse the extension. You will need to press and release the key again on a non-internal page for the extension to recognize that you have released the key.

FAQs

I broke something!

Tell me about it on Github. You can use Opera's addons site too, but Github is much easier for me to work with.

Can you make Ctrl+T open tabs next to the active tab, but the New Tab button open them at the end?

Nope. As far as Opera is concerned, new tabs are new tabs. I cannot differentiate between tabs opened with Ctrl+T, a mouse gesture, or the New Tab button.

Can you make <keyboard shortcut> do <cool feature>?

Maybe. Opera has implemented the chrome.commands API, but some functions (such as reopening closed tabs) aren't available to extensions and extensions can't assign commands to shortcuts like Ctrl+Z which already have default actions.

Can you make Ctrl+Tab cycle through tabs in last-visited order?

No, but Opera can. First, get Opera 16 or later. Then, go enter opera:flags in the address bar and look for the Tab cycling in activation order option.

Can you make <mouse command> on a tab do <cool feature>?

Unfortunately no. Opera extensions cannot tell when you click on any part of Opera's UI. Google Chrome has no such extension API either, so it is unlikely that this will be possible any time soon.

Can you add things to the tab context menu?

See above.
